Minecraft Custom Ore Generation

A guide to ore spawn locations and rarities in this world.

Important Notes

This wiki reflects **custom ore generation rules** implemented via datapacks. Default ore spawning has been disabled for most ores to ensure these rules apply.

Coal Ore: Coal still generates in ALL Overworld biomes as per default Minecraft behavior. No custom rules apply to coal.
